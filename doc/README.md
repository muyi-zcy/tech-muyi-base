# Tech-Muyi-Base 文档中心

本目录按“概览文档、快速开始文档、用户手册目录（按模块介绍）”三层组织，便于不同角色按需查阅。

## 概览文档

- `doc/概览文档.md`：项目定位、模块全景、阅读路径与维护原则。

## 快速开始文档

- `doc/快速开始文档.md`：环境要求、依赖接入、最小配置、首日落地清单。

## 用户手册目录（按模块介绍）

- `doc/用户手册/README.md`：模块手册总入口。

## 文档维护约定

- 文档命名采用语义化文件名，不使用序号前缀。
- 新增模块文档统一放在 `doc/用户手册/` 并在其 `README.md` 中登记。
- 跨文档引用统一使用完整路径（例如 `doc/用户手册/xxx.md`）。
- 配置项变更需补充默认值、是否必填、示例值和影响范围。
