# 概览文档

## 1. 项目定位

`tech-muyi-base` 是面向 Spring Cloud Alibaba 生态的企业级后端基础能力集合，采用多模块 Maven 聚合结构，支持按需组合能力。

## 2. 文档结构

当前文档分为三层：

- 概览文档：帮助快速理解项目全貌和阅读路径。
- 快速开始文档：指导从 0 到 1 完成接入。
- 用户手册目录：按模块介绍能力、配置、示例与常见问题。

## 3. 模块能力全景

- 基础能力：core/common/config/util/exception/id/on
- 数据与存储：redis/flyway/db-shardingjdbc/oss
- 微服务与通信：rpc/feign/mq/message/okhttp/sso/sentinel
- 任务与调度：easyjob/elasticjob
- 日志链路与观测：log/tracer/my-log-plugins
- 开发辅助：api/generator/dmask/boot/dependencies

## 4. 推荐阅读路径

1. 首次接入：先读 `doc/快速开始文档.md`。
2. 进入模块化使用：从 `doc/用户手册/README.md` 按业务能力选择文档。
3. 团队规范落地：优先阅读代码规范与生成器相关文档。

## 5. 维护原则

- 文档新增或变更时，同步更新 `doc/README.md` 与 `doc/用户手册/README.md`。
- 模块文档优先沉淀“可复制配置 + 最小示例 + 常见问题”。
